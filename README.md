# 🎙️ ScamVoice

> An AI-powered voice assistant that helps identify potential scam calls in real time and empowers users to make safer decisions.
## 🌍 Overview

ScamVoice was built during the **Google Hackathon** with one simple goal:

> **Protect people from voice-based scams before they become victims.**

Phone scams have become increasingly sophisticated. Fraudsters impersonate banks, government agencies, customer support, and even family members using AI-generated voices. ScamVoice acts as an intelligent layer of defense by analyzing conversations, identifying suspicious patterns, and providing real-time assistance to users.

Instead of reacting after a scam occurs, ScamVoice focuses on **prevention**.

      ▼

---

## 💡 Inspiration

Millions of people receive fraudulent calls every year.

Many victims are elderly users, first-time smartphone owners, or individuals unfamiliar with modern scam tactics. Existing spam filters mainly focus on text messages and emails, leaving voice communication relatively unprotected.

ScamVoice was created to bridge that gap by using AI to recognize warning signs during conversations and help users respond confidently.

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/sidcodes-07/ScamVoice-.git
```

### Navigate to the project

```bash
cd ScamVoice-
```

### Install dependencies

```bash
# Frontend
npm install

# Backend
pip install -r requirements.txt
```

### Start the application

```bash
npm run dev
```

or

```bash
python app.py
```
